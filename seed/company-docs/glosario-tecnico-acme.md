# Glosario técnico — Acme

| Término | Definición |
|---------|------------|
| AC-YYYY-NNN | Código interno proyecto Acme |
| FAT | Factory Acceptance Test en nave Acme |
| SAT | Site Acceptance Test en planta cliente |
| RFQ | Request for Quotation / solicitud presupuesto |
| Cambio de formato | Secuencia cambio utillajes/producto en línea |
| u/min | unidades por minuto (cadencia) |
| IP65 | Protección chorro agua (IEC 60529) |
| BOM | Bill of Materials, lista materiales |
| OT | Oficina técnica |
| BORRADOR | Estado documento no válido comercialmente |

## Abreviaturas materiales

- **S355:** acero estructural laminado
- **AISI 304/316L:** inox austenítico
- **PP/PET:** polímeros bandeja termoformada
